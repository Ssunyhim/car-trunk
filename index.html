이 페이지는 suv_data.json 파일을 자동으로 읽어옵니다. HTML은 손댈 필요 없이 JSON 파일만 수정하면 화면이 자동으로 바뀝니다.

{"brand":"현대","model":"팰리세이드","powertrain":"ICE","trunk":545,"trunkMax":1297,"seats":"7인승","image":""}

trunk = 기본 트렁크, trunkMax = 3열 폴딩 시 최대. powertrain 은 "ICE"(내연) / "EV"(전기). image 에 URL을 넣으면 사진, 비우면 일러스트.

※ 수치는 각 브랜드 플래그십 SUV의 공개 제원 기준(시장·트림에 따라 측정 방식 차이 가능).
기본 트렁크는 전 좌석 사용 시(3열 SUV는 3열을 세운 상태) 기준입니다. 실제 구매 전 제조사 공식 제원을 확인하세요.
